# student-result-analysis-project
# Student Result Analysis and Performance Visualization System

## Project Overview
- This project is created for **analyzing student results**.  
- Built using **Python, Pandas, and Matplotlib**.  
- You can enter a student’s **Roll Number** to see their detailed result and subject-wise performance chart.  
- You can also view **class performance** through charts for Topper, Average, Weak, and Fail students.  

---

## Features
- Load student data from **CSV file** (`students_50.csv`)  
- Automatically calculate:
  - Total Marks  
  - Percentage  
  - Pass/Fail  
- Search a student using **Roll Number**  
- Display student result in **vertical table format** (like a report card)  
- Show **subject-wise bar chart** for an individual student  
- Class charts for all students:
  - **Topper Students** (80% and above)  
  - **Average Students** (60–79%)  
  - **Weak Students** (40–59%)  
  - **Fail Students** (below 40%)  
- Easy to **update CSV** with more students  

---

## Technology Used
- **Python 3.x** → Programming language  
- **Pandas** → For data handling and calculations  
- **Matplotlib** → For creating charts and visualizations
